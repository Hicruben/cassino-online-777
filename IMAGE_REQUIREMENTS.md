## 图片处理规范

### 格式要求
- **WebP**: 主要格式,现代浏览器
- **JPG**: 降级备用格式
- **PNG**: Logo和需要透明背景的图片

### 尺寸规范
- **Hero**: 1920x600px (桌面), 750x500px (移动)
- **Cards**: 400x300px 或 350x250px
- **图标**: 150x150px 或 200x200px
- **Logo**: 200x60px (主), 150x45px (副)

### 优化要求
- WebP质量: 80%
- JPG质量: 85%
- 最大文件大小: Hero < 200KB, Cards < 50KB, Icons < 20KB
- 使用lazy loading

### 命名规范
```
hero-{page-name}.webp
hero-{page-name}.jpg (备用)
card-{category}-{number}.webp
icon-{name}.webp
logo-primary.webp
logo-secondary.webp
favicon-32.png
favicon-16.png
favicon.ico
```
